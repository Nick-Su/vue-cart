# vue-cart

## Project setup
1. Clone this repo. Open terminal and run
```
git clone https://github.com/Nick-Su/vue-cart.git
```
2. Open project folder
```
cd vue-cart-master
```
3. Install all dependencies by running the following command
```
yarn install
```
4. Then run
```
yarn serve
```
5. Now open your browser and visit 
```
http://localhost:8080/ 
```
